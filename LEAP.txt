       IDENTIFICATION DIVISION.
       PROGRAM-ID. LEAP.
       ENVIRONMENT DIVISION.
       DATA DIVISION.
       WORKING-STORAGE SECTION.
       01 WS-YEAR PIC 9(4).
       01 WS-RESULT PIC 9.
       01 WS-REMAINDER PIC 9999.
       01 WS-GIVING PIC  9(4).
       PROCEDURE DIVISION.
       LEAP.
       DIVIDE WS-YEAR BY 4 GIVING WS-GIVING REMAINDER WS-REMAINDER 
       IF WS-REMAINDER = 0 THEN 
          DIVIDE WS-YEAR BY 100 GIVING WS-GIVING REMAINDER WS-REMAINDER
         IF WS-REMAINDER = 0 THEN
          DIVIDE WS-YEAR BY 400 GIVING WS-GIVING REMAINDER WS-REMAINDER
          DISPLAY WS-REMAINDER
             IF WS-REMAINDER = 0 THEN
                SET WS-RESULT TO 1
                DISPLAY "DIVISIBLE ENTRE 100 Y 400 " WS-YEAR
             ELSE
                SET WS-RESULT TO 0
                  DISPLAY "NO DIVISIBLE ENTRE 400 " WS-YEAR
             END-IF
          ELSE
             SET WS-RESULT TO 1
             DISPLAY "DIVISIBLE ENTRE 4 PERO NO ENTRE 100 " WS-YEAR
          END-IF
       ELSE
          SET WS-RESULT TO 0
          DISPLAY " NO DIVISIBLE ENTRE 4 " WS-YEAR
       END-IF
         CONTINUE.
       LEAP-EXIT.
         EXIT.
